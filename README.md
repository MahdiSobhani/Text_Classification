# text_classification
Fine-tuning a BERT model and deploying it with Flask as a web page and API access.
In this app, the Persian model "demoversion/bert-fa-base-uncased-haddad-wikinli" has been fine-tuned to categorize three classes (sports, cinema, and literature), which the user can access in two ways: API or web page.
Not much data was used for training, but the output is interesting.
